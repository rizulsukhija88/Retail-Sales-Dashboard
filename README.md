# Retail Sales Performance Dashboard

## Project Overview

This project analyzes 131,706 retail sales records using Microsoft Excel. The objective was to transform raw transaction data into an interactive dashboard that helps users understand revenue performance, customer activity, product performance, and seasonal sales patterns.

## Dashboard Preview

![Retail Sales Dashboard](images/dashboard-overview.png)

## Key Performance Indicators

- **Total Sales:** $1,578,038.62
- **Total Transactions:** 64,682
- **Average Transaction Value:** $24.40
- **Unique Customers:** 22,625

## Dataset

The dataset contains retail transaction records from 2016, including:

- Transaction date
- Customer ID
- Transaction ID
- Product category
- Product SKU
- Quantity sold
- Sales amount

## Data Cleaning

The original CSV required several preparation steps before analysis:

- Corrected the date interpretation from text to valid Excel dates
- Converted sales amounts from text to numeric values
- Verified missing values and duplicate records
- Created helper columns for month, weekday, and weekday/weekend analysis
- Structured the dataset as an Excel Table for easier analysis and refresh

## Analysis Performed

The dashboard evaluates:

- Monthly sales trends
- Top-performing product categories
- Top-selling products by revenue
- Weekday versus weekend sales
- Sales performance by day of the week
- Overall customer and transaction activity

## Key Business Insights

### 1. Customer Purchase Frequency

The business generated 64,682 transactions from 22,625 unique customers, which is approximately 2.9 transactions per customer.

**Recommendation:** Introduce loyalty incentives and personalized promotions to encourage additional repeat purchases.

### 2. Average Transaction Value

The average transaction value was approximately $24.40, indicating that customers generally made relatively small purchases.

**Recommendation:** Use product bundles, cross-selling, and minimum-spend promotions to increase basket size.

### 3. Seasonal Sales Patterns

Monthly revenue varied throughout the year, showing that demand was not evenly distributed across all months.

**Recommendation:** Use historical sales patterns to improve inventory purchasing, promotional timing, and staffing decisions.

### 4. Product Revenue Concentration

A limited number of products and categories contributed a significant portion of total sales.

**Recommendation:** Maintain strong availability of high-performing products while reviewing weaker categories for pricing, promotion, or assortment changes.

### 5. Operational Planning

Weekday, weekend, and day-of-week analysis can help identify periods of stronger customer activity.

**Recommendation:** Align staffing levels and replenishment schedules with the highest-performing days.
